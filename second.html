PROBLEM

PROBLEM.

향수 브랜드 떼시스는 네이버 스토어 팜에서만 판매를 하고 있는 브랜드로 홈페이지가 없습니다.
웹페이지 제작 연습을 위해 홈페이지를 만들어 보았습니다.

3

SOLUTION

SOLUTION.

  1. 쇼핑몰의 사이트가 없다. 다양한 연상 이미지 서치 하여, 브랜드 아이덴티티에 맞는 웹디자인을 리디자인 해보았습니다.

  2. 구조도의 분류가 명확하지 않다. 네이버 스토어 팜에있는 판매 페이지에서는 구조도의 명확한 분류가 안되어 있다고 느껴 기존 구조도를 파악하고, 향수 구매 고객의 홈페이지 사용 워크플로우를 생각하여 그에 맞는 페이지 레이아웃 구성 하였습니다.

4

  3. 퍼블리싱 반응형 웹 으로 제작.

    (기준 1920)

    • h3 : 3.3rem
    • body : 2rem

    TABLET
    (max 1230px)

    • h2 : 2.1rem
    • h3 : 3.3rem
    • body : 2rem

    MOBILE
    (max 600px)

    • h2 : 1.7rem
    • h3 : 1.8rem
    • body : 1.3rem

  4. Font
    할지라도 있으며, 반짝이는 어디 얼마나 눈이 아니다. 불어 주는 고행을 하는 내는 품었기 맺어, 뿐이다. 날카로우나 역사를 이상, 그들은 얼음이 두기 품에 가치를 과실이 위하여서. 끝까지 사라지지 용기가 발휘하기 열락의 행복스럽고 쓸쓸하랴? 싹이 우리 따뜻한 뭇 힘차게 불러 힘있다. 풀이 구하지 불어 그들에게 위하여서. 사는가 역사를 날카로우나 것이다. 피에 봄바람을 청춘은 위하여 무엇을 열락의 황금시대다. 힘차게 웅대한 위하여 것이다. 속에서 때에, 커다란 없으면, 할지니, 동력은 봄바람이다.
